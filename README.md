# Programming Fundamentals & Algorithms – Academic Exercises

## Description
This repository contains a collection of small programming tasks and algorithmic exercises
created during Computer Science studies and individual practice.

The projects focus on understanding programming fundamentals, algorithms and problem-solving
rather than building production-ready applications.

---

## Project Scope
The repository includes examples related to:
- basic algorithms and data processing
- control structures and program logic
- simple problem-solving tasks
- understanding code correctness and readability

These exercises were developed for educational purposes as part of academic coursework
and personal learning.

---

## Technologies & Tools
- Python
- C / C++
- Basic programming and algorithmic concepts

---

## Learning Objectives
- Developing problem-solving and analytical thinking
- Understanding fundamental algorithms and programming logic
- Writing clear and readable code
- Building a foundation for further technical development

---

## Context
This repository was created as part of:
- Engineering studies in Computer Science
- Individual learning and practice of programming fundamentals

It represents an academic, junior-level collection of exercises.
